Etiquetas Semánticas Estructurales

div - Es una etiqueta estructural pero que no tiene semántica

header - Cabecera de un sitio web o de una sección

main - Define la sección principal del documento, sólo puede existir una etiqueta main por documento

footer - Pie de página de un sitio web o de una sección

nav - Representa una navegación

article - Representa un contenido dependiente de una sección

aside - Representa contenido complementario o secundario

section - Representa una sección de contenido genérico

address - Representa una información de contacto

Etiquetas de Bloque VS Etiquetas de Línea

La etiqueta de bloque por excelencia es la div, una etiqueta de bloque es aquella que genera saltos de línea con sus etiquetas hermanas.

La etiqueta de línea por excelencia es la span, una etiqueta de línea sólo ocupa el espacio necesario que tiene su contenido. Hola, soy otra span.

Normalmente se implementa en hojas de estilo externas vinculadas al documento HTML. En ocasiones podemos construirlo dentro de la etiqueta <style> en el head del documento o en línea dentro de la etiqueta.

Scripts en HTML

Para incluir JavaScript y dar funcionalidades al documento HTML usaremos la etiqueta <script>. Esta etiqueta puede ir en el head del documento o dentro del body. Si la incluimos en el body es preferible que esté al final del mismo.

La etiqueta <script> puede contener JavaScript o vincular un archivo externo de JavaScript.

Imágenes

En HTML podemos utilizar diferentes tipos de imagen dependiendo de su compresión. Sin pérdida de calidad: gif y png, o con pérdida de calidad como jpg.

GIF: formato sin pérdida con capacidad para comprimir hasta 256 colores, válido para tintas planas, logotipos, etc. Acepta animaciones.

PNG-8: Igual que GIF, pero su ratio de compresión es mejor. No acepta animaciones.

JPG: formato perfecto para cautricomías e imagen fotográfica, pero tiene pérdida de calidad.

PNG-24: formato perfecto para fotografías y transparencias a color real, tiene muy poca pérdida de calidad.

SVG: formato de imagen vectorial, sin pérdida, y la más recomendable para logotipos y pictogramas
